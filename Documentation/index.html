Introduction

Hey there! Thank you for your interest in JupiterMeet!

JupiterMeet is a room based audio/video meeting solution in pure WebRTC with a Signaling Server. It comes with a powerful admin panel written in Laravel.

The system doesn't use any third party services or APIs to host meetings. It is purely built upon native JavaScript APIs. JupiterMeet uses WebRTC technology so, the media will be end-to-end encrypted by default, which makes the system very secure.

JupiterMeet is divided into two parts
  1. A Web application in Laravel
  2. A Signaling server in NodeJS

Application Flow

JupiterMeet has a fully responsive design and rich UI, which makes it very easy to use in the mobile browsers too.

If auth mode is enabled, users can register, create meeting, send email invite and join the meeting. The moderator role is assigned to the meeting creator, he can allow or reject participant's request to join the meeting or kick them out of the meeting.

If auth mode is disabled, users can directly join the meeting without creating it first. (like the older version)

The admin can change the logo, application name, theme color etc from the portal itself. He can set a maximum meeting time limit as well.

Users can enjoy the built-in features such as chat, file sharing, screen sharing, picture-in-picture mode, full screen mode, one tap link sharing etc.

Note: For iOS (V11+) mobile devices, only Safari browser is supported.

Requirements

The system works in any server that has PHP and NodeJS installed

  • VPS or dedicated server
  • Web server & SSL
  • PHP >= 7.3
  • MySQL
  • NodeJS >= 12

Notes:

  • The installation manual will help you install all the dependencies
  • HTTPS is required
  • Shared hosting will not work (unless you host the server separately)

Installation

Get a bash script with the software, which will install and configure all the dependencies required to run JupiterMeet, followed by a step-by-step installation wizard! It will take care of everything in a few minutes in a clean Ubuntu 18.04 or CentOS 7 VPS.
